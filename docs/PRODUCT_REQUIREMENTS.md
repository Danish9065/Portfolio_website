# Product Requirements

One portfolio web app serves recruiters and freelance clients through shared content and clear audience paths.

Implemented:

- Public routes for home, about, projects, project detail, experience, services, resume, testimonials, and contact.
- Admin routes for login, dashboard, projects, inquiries, services, and testimonials.
- Backend-owned integrations for Supabase, Cloudinary, Resend, and Gemini.
- Editable demo content when Supabase is not configured.
- Honest disabled or partial-success states for missing credentials.
