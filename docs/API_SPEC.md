# API Spec

Base URL: `/api`

Public:

- `GET /health`
- `GET /profile`
- `GET /projects`
- `GET /projects/{id}`
- `GET /projects/slug/{slug}`
- `GET /skills`
- `GET /services`
- `GET /experience`
- `GET /testimonials`
- `POST /contact`
- `POST /chat`

Protected:

- `POST /uploads/image`
- `POST /uploads/resume`
- `POST /upload`
- `POST /admin/projects`
- `PUT /admin/projects/{id}`
- `DELETE /admin/projects/{id}`
- `GET /admin/inquiries`
- `POST /admin/services`
- `PUT /admin/services/{id}`
- `DELETE /admin/services/{id}`
- `POST /admin/testimonials`
- `PUT /admin/testimonials/{id}`
- `DELETE /admin/testimonials/{id}`
