# Design System

Tokens:

- Ink: `#080b12`
- Panel: `#101621`
- Elevated panel: `#151d2b`
- Line: `#263244`
- Text: `#d7dde8`
- Muted text: `#91a0b5`
- Accent: `#4dd7b0`
- Highlight: `#f4bf62`

Components use 6-8px radii, visible focus states, semantic HTML, responsive grid layouts, icon buttons where useful, and concise motion.
